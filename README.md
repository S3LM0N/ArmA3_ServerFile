## Getting Started / 어떻게 시작하나요?

본 레포에서는 암호화된 .pbo 형식의 파일만을 업로드합니다.
때문에 로컬에서(또는 편집기에서) 플레이는 어렵습니다.

(고용량, 매우 많은 양의 파일)

### Prerequisites / 선행 조건

2302 - 2306 포트를 UDP 프로토콜로 포트포워딩을 해야 합니다.
또한 방화벽 설정도 동일하게 해야 합니다.
DMZ는 필수는 아닙니다.

### Installing / 설치

.pbo파일을 업로드하기 때문에 데디케이트 서버 툴을 사용해서 서버를 실행시킬 수 있습니다.
